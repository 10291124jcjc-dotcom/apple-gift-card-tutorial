## 注意事项

- 美区 Apple 礼品卡只能兑换到美区 Apple ID。
- 第一次购买建议先用小额测试。
- 不要公开展示真实邮箱、订单号和礼品卡兑换码。
- 礼品卡通常不支持退款，购买前请确认地区和金额。
